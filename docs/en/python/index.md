# Python

MAVSDK provides two Python packages:

- **[mavsdk](api_reference/mavsdk/index.md)** — Synchronous bindings that call the C library directly (no gRPC dependency).
- **[aiomavsdk](api_reference/aiomavsdk/index.md)** — Asyncio wrapper around `mavsdk`, providing an async/await API similar to the original MAVSDK-Python.
